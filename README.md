# 🚀 Solidity Practice Repository

This repository contains a comprehensive collection of **Solidity smart contracts** covering fundamental to advanced concepts. It serves as a hands-on learning resource for understanding how smart contracts work on the Ethereum blockchain.

---

## 📌 Overview

The goal of this repository is to build a strong foundation in **smart contract development** by implementing and experimenting with various Solidity concepts.

The contracts range from:

* 🟢 Beginner concepts (variables, functions, loops)
* 🟡 Intermediate concepts (mappings, structs, inheritance)
* 🔴 Advanced concepts (assembly, delegatecall, create2, low-level calls)

---

## 🧠 Topics Covered

### 🔹 Basics

* Variables & Data Types (`Variables.sol`, `DefaultValues.sol`)
* Functions (`Functions.sol`, `FuncOutputs.sol`, `ViewAndPure.sol`)
* Control Flow (`IfElse.sol`, `ForAndWhile.sol`)
* Visibility (`Visibility.sol`)

### 🔹 Data Structures

* Arrays (`Array.sol`, `ArrayReplace.sol`, `ArrayShift.sol`)
* Mappings (`Mapping.sol`, `IterableMapping.sol`)
* Structs & Enums (`Structs.sol`, `Enum.sol`)

### 🔹 Smart Contract Concepts

* Constructors (`Constructor.sol`, `ParentConstructors.sol`)
* Inheritance (`Inheritance.sol`, `MultiInheritance.sol`)
* Interfaces (`Interface.sol`)
* Libraries (`Library.sol`)
* Modifiers (`FuncModifier.sol`)

### 🔹 Ethereum-Specific Features

* Payable & Ether Transfer (`Payable.sol`, `SendEth.sol`)
* Events (`Event.sol`)
* Global Variables (`GlobalVars.sol`)
* Error Handling (`Errors.sol`, `Unchecked.sol`)

### 🔹 Advanced Concepts

* Low-level Calls (`Call.sol`, `CallOtherContracts.sol`)
* Delegate Call (`DelegateCall.sol`)
* Fallback Functions (`FallBack.sol`)
* Function Selectors (`FunctionSelector.sol`)
* Encoding & Decoding (`EncodeCall.sol`, `ABIDecode.sol`)
* Create2 (`Create2.sol`)
* Assembly (`AssemblyVariables.sol`, `AssemblyIf.sol`, `AssemblyError.sol`)
* Bitwise Operations (`BitwiseOps.sol`)
* Contract Lifecycle (`Kill.sol`, `Immutable.sol`, `New.sol`)
